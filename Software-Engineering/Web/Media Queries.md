# Default Sizes

| Mode      | Device     | Default Size for Media Query |
| --------- | ---------- | ---------------------------- |
| Portrait  | Smartphone | 480px                        |
| Portrait  | Tablet     | 768px                        |
| Landscape | NoteBook   | 1024px                       |
| Landscape | Desktop    | 1200px                       |
| Landscape | TV         | >1200px                      |
